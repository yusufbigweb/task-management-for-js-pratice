# Task Management App (JavaScript Practice)

A simple Task Management Application built using **Vanilla JavaScript** to practice core concepts like DOM manipulation, event handling, and local storage.

---

## Features

- ✅ Add tasks  
- 📝 Edit tasks  
- ❌ Delete tasks  
- ✔️ Mark tasks as completed  
- 💾 Data stored in Local Storage  
- 🔄 Instant UI updates  

---

## Tech Stack

- HTML  
- CSS  
- JavaScript (Vanilla JS)
